# OS Abstraction Interfaces

[SSF](../README.md)

OS abstraction layer interfaces — primitives that wrap platform-specific threading,
synchronization, and scheduling so application code can stay platform-neutral.

| Module | Description | Source Files |
|--------|-------------|--------------|
| [ssflptask](ssflptask.md) | Low-priority background task queue (FIFO, single worker thread) | ssflptask.c, ssflptask.h |
